#+TITLE: ClusterdOS

*ClusterdOS* is a *distributed Operating System*, or '*dOS*' for short.

that bootstraps, maintains, and upgrades a fresh Kubernetes cluster smoothly and with minimal effort. It establishes a simple framework for adding and versioning new cloud-native applications which the dOS user can enable and configure with simple high-level helm parameters.

The hope of this dOS concept is to usher in the advent of the '*Personal Cluster*' as our next phase of computer use - following the evolutionary pattern of QDOS and the 'personal computer'

ClusterdOS can be used as an upstream for your own *distributed OS distros*. You can set ~targetRevision~ in ~install.yaml~ to freeze on a stable release and upgrade periodically.

If you would like your cluster to auto-update instead, you can point at the ~main~ branch. While best efforts will be made to avoid pushing breaking changes to ~main~, please keep in mind this is a bleeding-edge target and if your cluster is running critical production services it is certainly recommended to point to a stable tagged release instead.

* Prerequisites

1) A Kubernetes cluster

#+begin_src shell
sudo snap install microk8s
#+end_src


2) ArgoCD

#+begin_src shell
microk8s enable community
microk8s enable argocd
#+end_src

 
* Install ClusterdOS

#+begin_src shell
kubectl apply -f install.yaml
#+end_src
